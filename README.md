# CKEditor5 paste link

CKEditor 5 plugin that links the selected content with an URL from the clipboard.

## Developing the package

### Preview

```
yarn install
yarn start
```

This will open http://localhost:8080/ sample page with the plugin enabled.

### Testing

```
yarn install
yarn test -w
```
